# Google Maps Route Tracking

CC106 Lesson 10 Activity: Google Maps API – Destination Selection and Route Tracking

## Description
This project allows users to search for a destination and display the driving route from their current location using the Google Maps API.

## How to Run
1. Download the project files.
2. Open `index.html` in a web browser.
3. Allow location permission.
4. Search for a destination and click **Get Route**.
